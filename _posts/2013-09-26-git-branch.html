使用Git Bash和Git GUI工具实际操作

一，开发分支

一般有两个分支，默认主分支master，日常开发分支develop

从主分支master创建开发分支develop

git branch develop master

查看当前所有分支：git branch -a

切换分支：git checkout develop

在分支操作后，添加修改文件，未进行add跟踪和提交，直接切换分支会看到如下错误

图：branch1

需要commit后才可以切换分支
